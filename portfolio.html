PROGRAMS I BUILT

Swap-meet is a backend program built using OOP and test driven development to organize a swap meet. For the swap meet, each participant registers as a vendor online. Each vendor then creates an inventory of items they wish to swap. The program checks to see if the items a user wants to trade matches with another vendors item they wish to trade. If the two vendors items match, a trade is successful.

Task-list API is a web API built using RESTful design practice and CRUD methods using Flask package. Task-list API allows the user to create goals. The user identifies tasks associated with each goal. When a task is completed, the user marks the task as completed. When a task is marked completed, the API uses an external slack API that uses a slackbot to notify the user on slack that a task has been completed.

Retro Video Store is an API to handle the backend of a video store. This API works with the Flask package using RESTful design practices and CRUD methods. This API allows the user to create a database of customers and video inventory. It allows the user to check videos out to a customer. The API also handles when videos are checked back in. The API adjusts the inventory accordingly.
