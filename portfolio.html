Retro Video Store

The goal of this project was to design and deploy an API as well as a CLI for a hypothetical "Blockbuster"-style video store where clients would be able browse and rent videos, and video store employees would be able to update the database of customers and rentals.

Skills assessed:
  • Demonstrated understanding of the client-server model, request-response cycle and conventional RESTful routes.
  • Working with the Flask package:
    • Creating models
    • Creating conventional RESTful CRUD routes for models
    • Reading query parameters to create custom behavior
    • Create unconventional routes for custom behavior
    • Creating a many-to-many relationship between two models
    • Building a front-end Command Line Interface using an external API
