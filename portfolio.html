RETRO VIDEO STORE API

This project was aimed at creating a basic API to keep track of a rental video store. Attached to a postgres database, the API allows the user to add and delete video inventory and customers as well as check videos in and out to customers.

In this project: Python, Flask, Postgres -- deployed to Heroku
